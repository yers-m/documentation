# Handling BLLQ data
* Below Lower Limit of Quantification
* not always possible to improve the LLOQ of the bioanalytical assays [[ðŸ“–]](https://doi.org/10.1007/s10928-008-9094-4 "Ahn, 2008")
* ignoring data below LLOQ [[ðŸ“–]](https://doi.org/10.1023/a:1012299115260 "Beal, 2001")
    * biased and slightly overpredicted near or below LLOQ  
* BLQ observations should be considered if the fraction BLQ is larger than 10% [[ðŸ“–]](https://doi.org/10.1007/s10928-007-9078-9 "Byon, 2007")
* censoring BLQ in data with a low fraction BLQ (10%) [[ðŸ“–]](https://doi.org/10.1007/s10928-011-9201-9 "Xu, 2011")
    * 1-CMT model: minor impact
    * 2-CMT model: more impact
* 4 types of methods
    * the improvement depends on model complexity and fraction of observations below LLOQ [[ðŸ“–]](https://doi.org/10.1023/a:1012299115260 "Beal, 2001")
